# PROJECT — وضعیت زنده پایان‌نامه

> این سند زمینهٔ عملیاتی جاری است و باید پس از هر تغییر معنادار به‌روزرسانی شود.

## هویت

- پژوهشگر: شقایق امجدی خراسانی
- مقطع/رشته: کارشناسی ارشد ارتباط تصویری
- دانشگاه: دانشگاه تهران، دانشکده هنرهای زیبا
- عنوان نظری: پژوهشی پیرامون کدنویسی خلاق و بازتعریف نقش طراح گرافیک
- پروژه عملی: طراحی تصاویر متحرک مبتنی بر کد
- استاد راهنمای نظری: مصطفی گودرزی
- استاد راهنمای عملی: روح‌الله موحدی

## پرسش محوری کاری

کدنویسی خلاق چگونه می‌تواند نسبت طراح گرافیک با ابزار، فرایند، مادهٔ طراحی، سیستم، مؤلف‌بودن و خروجی را تغییر دهد؟ آیا این تغییر پارادایمی است، ابزاری است، یا مرز این دو را مبهم می‌کند؟

## موضع فعلی

- نتیجه از پیش تعیین نشده است.
- «طراح به‌مثابه برنامه‌نویس» فقط یکی از امکان‌هاست، نه نتیجهٔ مفروض.
- میان استفاده از نرم‌افزار آماده، نوشتن کد، طراحی سیستم و خلق متامدیا باید تمایز گذاشته شود.
- پروژه نظری و عملی باید از طریق مصداق‌ها و ثبت تأثیر متقابل به هم متصل شوند.

## چارچوب‌های نظری فعال

- مارشال مک‌لوهان: تغییر مقیاس، سرعت و الگو
- ویلم فلوسر: apparatus/program و امکان بازی یا خروج از برنامه
- لو مانوویچ: نرم‌افزار فرهنگی، metamedium و نسبت تألیف/استفاده
- ریچارد سنت: مهارت، ساختن و رابطه دست/ذهن
- مطالعات طراحی و آموزش: برنامه‌نویسی به‌عنوان طراحی، craft و creative coding

این فهرست قطعی نیست و ورود چارچوب تازه نیازمند توضیح کارکرد آن است.

## دارایی‌های فعلی ریپو

- مجموعه‌ای از کتاب‌ها و مقالات تبدیل‌شده به Markdown
- آرشیو آثار مرجع و آثار عملی در `archive/`
- سه منبع حقیقت ساختاریافته:
  - `archive/works-index.csv`
  - `archive/references-index.csv`
  - `archive/inspiration-map.csv`
- الگوی تحلیل آثار در `archive/reference-note-template.md`

## وضعیت آرشیو

برای آمار دقیق همیشه خود CSVها و `archive/README.md` بررسی شوند. هیچ عددی از حافظه گزارش نشود.

## تصمیم‌های پذیرفته‌شده

- شناسه آثار خودی: `CC-##`
- شناسه آثار مرجع: `RF-###`
- شناسه حذف‌شده دوباره استفاده نمی‌شود.
- آثار کنارگذاشته‌شده حذف نمی‌شوند و وضعیت آن‌ها ثبت می‌شود.
- لینک بدون فایل پشتیبان، آرشیو کامل نیست.
- تحلیل تفصیلی برای آثاری نوشته می‌شود که نقش «مصداق تحلیلی» دارند.

## تصمیم‌های نیازمند تثبیت

- صورت نهایی پرسش اصلی و پرسش‌های فرعی
- ساختار فصل‌ها و زیربخش‌ها
- محدوده دقیق «طراح گرافیک» در برابر creative technologist / computational designer
- معیار انتخاب مصادیق تحلیلی
- سیاست استفاده و بازنشر تصاویر آثار دیگران در نسخه دانشگاهی و انتشار عمومی
- واژه‌نامه نهایی فارسی/انگلیسی

## کارهای باز

- تطبیق منابع موجود با فصل‌ها و پرسش‌های فرعی
- ساخت ایندکس فیش‌های نظری
- تکمیل اطلاعات کتاب‌شناختی و شماره صفحات منابع اصلی
- تکمیل ردیف‌های آثار عملی و بک‌آپ‌ها
- ثبت رابطه هر اثر مرجع با پروژه عملی در `inspiration-map.csv`
- تهیه ماتریس ادعا ← شاهد ← منبع ← فصل
