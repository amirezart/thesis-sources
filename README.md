# Thesis Sources — Shaghayegh Amjadi

مخزن منابع، فیش‌ها، آرشیو آثار و زمینهٔ عملیاتی پایان‌نامهٔ کارشناسی ارشد ارتباط تصویری.

## موضوع

- **عنوان نظری:** پژوهشی پیرامون کدنویسی خلاق و بازتعریف نقش طراح گرافیک
- **پروژهٔ عملی:** طراحی تصاویر متحرک مبتنی بر کد
- **دانشگاه:** دانشگاه تهران، دانشکده هنرهای زیبا

پرسش پژوهش باید باز بماند: کدنویسی خلاق ممکن است نقش طراح را به‌صورت پارادایمی تغییر دهد، صرفاً ابزارهای او را توسعه دهد، یا مرز این دو را مبهم کند.

## منابع حقیقت

1. فایل اصلی/نسخه معتبر هر منبع
2. داده‌های ساختاریافته و فیش‌های دارای ارجاع
3. `archive/*.csv` برای آثار و روابط تأثیر
4. `docs/ai-context/PROJECT.md` برای وضعیت و تصمیم‌های جاری
5. گفتگوها و حافظهٔ دستیار فقط به‌عنوان زمینهٔ کمکی

در صورت تعارض، منبع بالاتر مقدم است و تعارض باید ثبت شود.

## قواعد فوری

- هیچ نقل‌قول، شماره صفحه یا اطلاعات کتاب‌شناختی بدون بررسی ساخته نشود.
- فایل منبع خام بدون ضرورت بازنویسی نشود؛ تحلیل در فایل جدا ثبت شود.
- تغییرات مهم، تصمیم‌ها و کارهای باز در `PROJECT.md` به‌روز شوند.
- برای آثار، قواعد `archive/README.md` لازم‌الاجراست.
- قبل از ادعای Git، وضعیت واقعی شاخه، کامیت و remote بررسی شود.
