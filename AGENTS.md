# Agent Entry Point

این ریپو مرجع عملیاتی پایان‌نامهٔ شقایق امجدی خراسانی است.

پیش از هر کار معنادار، به‌ترتیب زیر بخوان:

1. `README.md`
2. `docs/ai-context/PROJECT.md`
3. سند تخصصی مرتبط در `docs/ai-context/`
4. برای آرشیو آثار: `archive/README.md` و راهنماهای همان پوشه

قواعد کلیدی:
- حقیقت ریپو بر حافظهٔ گفتگو مقدم است.
- نتیجهٔ پژوهش از پیش تعیین نمی‌شود.
- نقل‌قول، صفحه یا دادهٔ تأییدنشده ساخته نمی‌شود.
- وضعیت‌ها و تصمیم‌های مهم در `PROJECT.md` ثبت می‌شوند.
- هیچ ادعایی دربارهٔ commit/push/merge بدون راستی‌آزمایی مجاز نیست.
